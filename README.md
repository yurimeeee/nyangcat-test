## 👀프로젝트 소개
* React로 구현한 MBTI 테스트로, "내가 갑자기 고양이가 된다면.."이라는 테마를 가진 테스트입니다.</br>
카카오톡 공유 기능으로 지인과 함께 간단하게 즐길 수 있습니다.
